# Legacy Scripts

This folder keeps one-off maintenance or investigation scripts that are no longer part of the active Fries build and release flow.

- `probe-openai.mjs`: historical Playwright probe for inspecting ChatGPT and team-panel sessions
- `probe-team-network.mjs`: historical network probe for the old team panel workflow
- `fix_windows_icons.ps1`: one-off Windows shortcut and icon-cache repair helper

They are kept for reference only and should not be treated as required runtime or release scripts.
