# Fries v0.4.6-beta

This beta expands the public release page into a fuller multi-platform download archive.

## Highlights

- Added Linux assets to the official GitHub Release flow: `AppImage`, `deb`, and `rpm`.
- Updated the download guidance so Windows, macOS, and Linux users can quickly pick the right installer/package.
- Continued to keep macOS in unsigned beta mode while preserving the signing/notarization path for a future signed release.

## Platform Notes

- Windows ships as installer, portable, and unpacked directory.
- Linux now ships as `AppImage`, `deb`, and `rpm`.
- macOS continues to ship as an **unsigned beta** `zip / dmg`.
