# Fries v0.4.5-beta

This beta focuses on heatmap readability and release alignment.

## Highlights

- Added a settings toggle to switch heatmap thresholds between `Auto` and `Fixed`.
- Added a clearer “Current mode” hint above heatmap views so users can tell which scaling mode is active.
- Refreshed release metadata and documentation so local packaging, GitHub Releases, and UI version strings stay aligned.

## Platform Notes

- Windows ships as installer, portable, and unpacked directory.
- macOS continues to ship as an **unsigned beta** `zip / dmg`.
- Linux is still not an official release target.
